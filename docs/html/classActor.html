Detailed Description

template<typename PolicyType, typename StateType, typename ActionType>
class Actor< PolicyType, StateType, ActionType >

The Actor class represents an agent that selects actions in a given environment.

This class uses a specified policy to select actions based on an environment's current state. It readily offers uniform random action-selection without further specification to enable epsilon-greedy exploration. It additionally houses a random number generator, which is to be provided by an instance of the RandomEngine class in order to enable global seeding and reproducibility of results across multiple agents. This class currently does not support multi-dimensional action spaces or action space representations with varying size that vary from state to state.

Template Parameters
