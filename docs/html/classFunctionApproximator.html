Detailed Description

template<typename Derived, typename PredictionType, typename StateType>
class FunctionApproximator< Derived, PredictionType, StateType >

Base class providing a unified interface for various function approximators.

The FunctionApproximator class serves as a flexible wrapper to standardize the interface for different types of function approximators. This allows users to integrate models ranging from simple linear regressions implemented in Eigen to complex deep neural networks built with libraries like LibTorch, all within a consistent API. The primary purpose of this class is to ensure that any function approximator, regardless of its underlying implementation, adheres to a common structure, facilitating seamless integration with the rest of the AsyncRL++ library. Use this class template for any policy-function or value-function approximators.

Template Parameters
DerivedThe specific function approximator class inheriting from this base class.
PredictionTypeThe type of the output produced by the function approximator (must have a .size() method).
StateTypeThe type representing the input state used for predictions.
