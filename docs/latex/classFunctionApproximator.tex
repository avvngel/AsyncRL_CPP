\subsubsection*{template$<$typename Derived, typename Prediction\+Type, typename State\+Type$>$\newline
class Function\+Approximator$<$ Derived, Prediction\+Type, State\+Type $>$}

Base class providing a unified interface for various function approximators. 

The \mbox{\hyperlink{classFunctionApproximator}{Function\+Approximator}} class serves as a flexible wrapper to standardize the interface for different types of function approximators. This allows users to integrate models ranging from simple linear regressions implemented in Eigen to complex deep neural networks built with libraries like Lib\+Torch, all within a consistent API. The primary purpose of this class is to ensure that any function approximator, regardless of its underlying implementation, adheres to a common structure, facilitating seamless integration with the rest of the Async\+RL++ library. Use this class template for any policy-\/function or value-\/function approximators.
